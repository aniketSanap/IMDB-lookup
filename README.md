# IMDB-lookup
Find IMDB rating of any movie/show from the command line

#### DEPENDENCIES:
bs4: \
`pip install bs4` \
python 3.x \
An internet connection

#### USAGE:
On GNU/Linux \
`$ ./imdb.py <name of show/movie>` or `python3 imdb.py <nameof show/movie>`
Example: \
`$ ./imdb.py fight club` \
On windows: \
`python3 imdb.py fight club`
